# Test Scenarios

## TS-001: Home Page
Verify that the DemoBlaze home page loads successfully and main navigation elements are visible.

## TS-002: Product List
Verify that product cards are displayed correctly with product images, names, prices and descriptions.

## TS-003: Product Categories
Verify that the user can filter products by Phones, Laptops and Monitors.

## TS-004: Category Pagination
Verify that pagination buttons behave correctly when a product category is selected.

## TS-005: Product Details
Verify that the user can open a product detail page and view product name, image, price and description.

## TS-006: Add to Cart
Verify that the user can add a product to the cart from the product detail page and receives a confirmation message.

## TS-007: Cart Page
Verify that added products appear correctly in the cart with product image, title, price, delete option and total price.

## TS-008: Remove Product from Cart
Verify that the user can remove a product from the cart.

## TS-009: Cart Total Calculation
Verify that the cart total price is calculated correctly when multiple products are added.

## TS-010: Place Order Modal
Verify that the Place Order modal opens correctly and displays all required checkout fields.

## TS-011: Complete Order with Valid Data
Verify that the user can complete the checkout form with valid filled data and receive an order confirmation.

## TS-012: Checkout Field Validation
Verify that the checkout form handles empty required fields correctly and prevents order completion when required data is missing.

## TS-013: Payment Field Validation
Verify that payment-related fields such as Credit card, Month and Year validate input format correctly.

## TS-014: Empty Cart Checkout
Verify that the user cannot complete an order when the cart is empty.

## TS-015: Sign Up Modal
Verify that the Sign up modal opens correctly and displays username and password fields.

## TS-016: Valid Sign Up
Verify that a new user can sign up with a unique username and password.

## TS-017: Invalid Sign Up
Verify that the system handles empty or invalid sign up credentials correctly.

## TS-018: Log In Modal
Verify that the Log in modal opens correctly and displays username and password fields.

## TS-019: Valid Log In
Verify that a registered user can log in with valid credentials and see the logged-in state.

## TS-020: Invalid Log In
Verify that the system displays an error message when invalid login credentials are used.

## TS-021: Log Out
Verify that a logged-in user can log out successfully and return to the logged-out navigation state.

## TS-022: Contact Form
Verify that the Contact form opens correctly and allows the user to send a message with filled fields.

## TS-023: Contact Form Validation
Verify that the Contact form prevents submission when required fields are empty.

## TS-024: About Us Modal
Verify that the About us modal opens correctly and displays available video/content.

## TS-025: Navigation Back to Home Page
Verify that the user can return to the home page from another page using the Home navigation link.
