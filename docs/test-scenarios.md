# Test Scenarios

## TS-001: Home Page
Verify that the DemoBlaze home page loads successfully and main navigation elements are visible.

## TS-002: Product Categories
Verify that the user can filter products by Phones, Laptops and Monitors.

## TS-003: Product Details
Verify that the user can open a product page and view product name, price, image and description.

## TS-004: Add to Cart
Verify that the user can add a product to the cart from the product detail page.

## TS-005: Cart Page
Verify that added products appear correctly in the cart.

## TS-006: Remove Product from Cart
Verify that the user can remove a product from the cart.

## TS-007: Place Order
Verify that the user can complete the order form with valid data.

## TS-008: Checkout Validation
Verify that checkout behavior is handled correctly when required fields are empty.

## TS-009: Sign Up
Verify that a new user can sign up with valid username and password.

## TS-010: Invalid Sign Up
Verify that the system handles empty or already existing sign up credentials.

## TS-011: Log In
Verify that a registered user can log in with valid credentials.

## TS-012: Invalid Log In
Verify that the system displays an error when invalid login credentials are used.

## TS-013: Log Out
Verify that a logged-in user can log out successfully.

## TS-014: Contact Form
Verify that the user can send a message through the contact form.

## TS-015: Basic UI Check
Verify that buttons, links, modals and navigation elements are displayed clearly.
